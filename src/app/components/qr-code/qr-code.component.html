¿Cómo usar el Generador de Código QR? (Haz clic para expandir)

1. Escribe o pega cualquier enlace (URL), texto plano o número de contacto en el campo de entrada.

2. El código QR se generará de manera instantánea y en tiempo real.

3. Utiliza el botón "Descargar PNG" para guardar el código en tu dispositivo, o presiona "Compartir" para enviarlo mediante las redes compatibles de tu sistema.
