¿Cómo usar el Compresor de Imágenes? (Haz clic para expandir)

1. Selecciona o arrastra una o varias imágenes usando el selector de archivos.

2. Ajusta el nivel de Calidad mediante la barra deslizadora para controlar la compresión en tiempo real.

3. Utiliza el comparador deslizante visual para inspeccionar el cambio de calidad de la imagen seleccionada.

4. Presiona el botón "Descargar" para empaquetar y descargar todas las imágenes optimizadas en un archivo ZIP.
